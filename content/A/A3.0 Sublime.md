# Sublime
Focus: [[F3.0 Architecture|Architecture]]
Skills used: [[T2.0 Rhino|Rhino3d]], Unreal Engine 5, and [[T9.0 Photoshop]]
## Introduction
My vision was to create a space for mental, physical and spiritual well-being. The circular path around the building is intended to act as a ritualistic space where one can engage with the spirits of the place. The surrounding nature allows for mental well-being. Finally, the large, biophilic space, including the long weaving pathway, encourages physical activity and interaction with the space. The minimalistic yet dynamic building at the center is humble in shape and style, and intends to direct the viewer to the sky and mountains around it.
### Goals
The goal of this project was to design a concept of a space that would be beneficial for mental and physical health.  Since this was a personal project, the goal of the project changed throughout.
![](ASW_MHS-Rhino3.png)
At first, the building was surrounding the winding pathway which would encircle a courtyard. Since the building was too big, I changed the layout so that the building would go inside the rounded pathway.
## Materials and Methods
I started off by drawing a concept for a building. I imagined the floor plan being un-filled squares that would rotate 45 degrees every 3rd floor. Afterwards, I would add the facade or the building enclosure which would wrap itself around the dynamic 3D shape. I went straight into Rhino3D to learn the commands, to learn how to use SubD's and Nurbs Curves and how to edit the curved planes I created.
![](DART_MHS.png)
##### Limitations
One of the hardest parts of completing this project was transferring such a huge Rhino file to Unreal Engine 5. It required me watching multiple YouTube videos, importing my project in different portions, and re-editing my model after some portions of my building wouldn't show up, and Unreal Engine 5 would give me multiple errors.
![](ASW_MHS-Rhino2.png)
## Results
Using the concept i made in Rhino, I started making a Revit file of a building with similar form and size. The Revit file is was made using in-place masses, creating multiple mullion-lines, and creating an expanded axonometric drawing.
